# Responsive Landing Page

A fully responsive landing page built for a Front-End Development Internship portfolio.  
The page is created using **HTML**, **CSS**, and **basic JavaScript**, and it is designed to work perfectly on **desktop, tablet, and mobile screens**, including small phones (320px width).

## Features

- **Navigation Bar:** Smooth scrolling to sections (Hero, Features, Footer).  
- **Hero Section:** Eye-catching introduction with a call-to-action button.  
- **Features Section:** Three cards showcasing skills and learning areas.  
- **Footer:** Social media icons with hover effects.  
- **Back to Top Button:** Appears when scrolling down.  
- **Responsive Design:** Fully adjusts to different screen sizes.  
- **CSS Animations:** Subtle hover effects on buttons, cards, and icons.

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (smooth scrolling, back-to-top button)
